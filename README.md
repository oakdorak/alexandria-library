# 🧬 Biblioteca de Alejandría

> Archivo público de síntesis científica y soberanía tecnológica compilado autónomamente por los bucles del Daemon Odysseus.

Un espacio de documentación e investigación en ciencias cognitivas, autonomía biológica, procesamiento neuronal local e ingeniería de sistemas local-first. Diseñado con una estética táctica oscura y libre de telemetría o dependencias propietarias.

---

## 🚀 Despliegue Local

Puedes clonar el archivo para visualizar la biblioteca localmente:

```bash
git clone https://github.com/oakdorak/alexandria-library.git
cd alexandria-library
# Abre index.html directamente en tu navegador o levanta un servidor estático:
python3 -m http.server 8000
```

## 🧬 Bucle de Destilación Autónomo (Odysseus Hook)

La base de datos de esta biblioteca (`data/catalog.json`) se regenera y actualiza automáticamente mediante los ciclos de **Odysseus** corriendo en el Nodo Beta (NixOS). Cada ciclo:

1. **Investiga:** Consulta repositorios científicos públicos (como OpenAlex) para profundizar en las vertientes de estudio de la tripulación.
2. **Sintetiza:** Traduce y condensa los hallazgos en reportes de aprendizaje estructurados.
3. **Indexa:** Genera matrices de relación y similitud local de forma completamente offline.
4. **Despliega:** Compila la base estática y actualiza el repositorio en GitHub Pages.
